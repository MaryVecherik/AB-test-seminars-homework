# A/B тестирование

Учебный репозиторий по курсу **A/B тестирования и продуктовой аналитики**.

В репозитории собраны материалы семинаров и домашние задания, посвящённые анализу экспериментов, статистическим тестам и принятию решений на основе данных.

---

## 📌 О чём этот репозиторий

A/B тестирование — это метод сравнения двух вариантов (A и B), позволяющий определить, какой из них работает лучше по заданной метрике.

В рамках курса рассматриваются:
- постановка гипотез;
- выбор метрик;
- анализ результатов эксперимента;
- статистическая проверка гипотез;
- интерпретация результатов и принятие решений.

---

## 📊 Что изучается

### Seminar 5  
**Применение математической статистики в реальных задачах**

- применение статистических критериев к различным метрикам;
- анализ конверсий;
- проверка статистической значимости.

**Домашнее задание:**
1. Оценка значимости различий конверсии (A/B тест, alpha = 5%)  
   - расчёт и выбор статистического критерия;  
   - формулировка выводов и рекомендаций.

2. Выбор статистического критерия для сравнения конверсий при фиксированном размере выборки.

---

### Seminar 6  
**Расчёт длительности A/B теста и продвинутые темы**

- расчёт размера выборки;
- влияние alpha и beta;
- оценка минимального детектируемого эффекта (MDE).

**Домашнее задание:**
1. Расчёт длительности теста для изменения time spent (mail.ru).  
2. Расчёт длительности теста для изменения bounce rate в e-commerce.

### Seminar 7  
**Нестандартные ситуации в A/B тестировании**

- типичные ошибки продуктовых команд;

**Домашнее задание:**
- разработка плана A/B теста без peeking problem;
- выбор и описание метрик анализа.
